# 衛星データ活用計画 - 建設コスト予測への応用

## 概要

Sentinel-2 L2A（12バンド）と VIIRS（夜間光）を使用して、`construction_cost_per_m2_usd` に関連する特徴量を構築する。
以下に仮説ベースで活用案を整理する。

---

## 仮説と特徴量候補

### 仮説 1: 都市化・建設物密度が高いほど建設コストが高い

**根拠:** 都市部では地価・労働コストが高く、建設規制も厳しいため単価が上昇する。
コンクリート・アスファルト・金属屋根はSWIRで高反射、NIRで低反射という特性を持つ。

**特徴量候補:**

```
NDBI (Normalized Difference Built-up Index)
= (B11 - B8) / (B11 + B8)
```
- 正値 → 建設物・人工構造物が多い
- 負値 → 植生・水域

```
UI (Urban Index)
= (B7 - B4) / (B7 + B4)
```

```
IBI (Index-based Built-up Index)
= [NDBI - (NDVI + MNDWI) / 2] / [NDBI + (NDVI + MNDWI) / 2]
```
- NDVIとMNDWIで植生・水域の影響を除去した都市化指数

**集計方法:** 画像全体の平均・中央値・標準偏差・パーセンタイル (25, 75, 90)

---

### 仮説 2: 植生が少ない（都市的土地利用）ほど建設コストが高い

**根拠:** 緑地・農地は建設コストが低い地域に多く、植生量は都市化の逆指標となる。

**特徴量候補:**

```
NDVI (Normalized Difference Vegetation Index)
= (B8 - B4) / (B8 + B4)
```
- 低値 → 裸地・人工物・都市
- 高値 → 健全な植生

```
EVI (Enhanced Vegetation Index)
= 2.5 * (B8 - B4) / (B8 + 6*B4 - 7.5*B2 + 1)
```
- 都市部・高反射エリアでNDVIより安定

**活用方法:** NDVI 低値ピクセルの割合（閾値 0.2 以下）を都市化率として使用

---

### 仮説 3: 裸地・建設中エリアが多い地域は需要が高く建設コストが上昇する

**根拠:** 建設中の更地・土工事現場はSWIRで高反射、Blueで中反射という特性を持つ。裸地率が高い地域は開発需要が旺盛で建設コストが上昇傾向にある。

**特徴量候補:**

```
BSI (Bare Soil Index)
= ((B11 + B4) - (B8 + B2)) / ((B11 + B4) + (B8 + B2))
```
- 正値 → 裸地・建設工事現場

```
NBSI (Normalized Bare Soil Index)
= (B11 - B8) / (B11 + B8)  ※ NDEIとも呼ばれる
```

**活用方法:** BSI 高値ピクセル（BSI > 0）の割合

---

### 仮説 4: 水域・湿地に近い地域は基礎工事コストが増大する

**根拠:** 軟弱地盤・水害リスク地域では杭基礎・防水工事が必要となりコストが増す。

**特徴量候補:**

```
NDWI (Normalized Difference Water Index)
= (B3 - B8) / (B3 + B8)
```

```
MNDWI (Modified NDWI)
= (B3 - B11) / (B3 + B11)
```
- MNDWI は建設物と水域の混在時に NDWI より精度が高い

**活用方法:** MNDWI 高値ピクセル割合（水域率）、水域との距離の代替指標

---

### 仮説 5: 夜間光の強度が高い地域は経済活動が活発で建設コストが高い

**根拠:** VIIRS の avg_rad（平均夜間放射輝度）は電力消費量・経済活動と相関。
既に CSV に `deflated_gdp_usd` があるが、より細粒度の空間情報として補完できる。

**特徴量候補（VIIRS）:**
- `avg_rad` の平均・中央値・最大値・標準偏差・90パーセンタイル
- `avg_rad` の高値ピクセル割合（経済集積地の面積代替）

---

### 仮説 6: 地質・土壌の種類が基礎工事コストに影響する

**根拠:** 粘土質・軟弱地盤では杭基礎が必要。SWIRバンドの比率で鉱物組成が識別できる。

**特徴量候補:**

```
Clay Minerals Ratio = B11 / B12
```
- 粘土鉱物（カオリナイト等）に感度

```
Iron Oxide Ratio = B4 / B2
```
- 酸化鉄（赤褐色土壌）に感度

```
Carbonate Index = B11 / B8A
```

**活用方法:** 各比率の地域平均・標準偏差

---

### 仮説 7: 大気・エアロゾル量が建設環境（工業地帯）の代替指標になる

**根拠:** 工業地帯ではエアロゾル・粒子状物質が多く、B1（443nm）に感度が高い。
工業集積地は建設コストに独自の影響（特殊設備・環境規制）を与える可能性がある。

**特徴量候補:**
- `B1` の平均値（エアロゾル光学的厚さの代替）
- `B1 / B4` 比（大気散乱の影響度）

---

### 仮説 8: 画像テクスチャが建物密度・都市構造を反映する

**根拠:** 建物が密集する都市部では画像のテクスチャが複雑になる。
同じ NDBI 値でも、テクスチャが複雑な方が建設コストが高い傾向がある（高密度市街地）。

**特徴量候補:**
- 各バンド（特に B8, B11）の **標準偏差**（局所的な分散 = テクスチャの粗さ）
- **GLCM（Gray-Level Co-occurrence Matrix）特徴量**: Contrast, Homogeneity, Entropy
  - 計算コストが高いため、代替として `画像全体の標準偏差` を使用可能

---

### 仮説 9: 各バンドの生の統計値が非線形的に建設コストと関係する

**根拠:** 指数化しなくても、バンドの統計量そのものが土地被覆の情報を持つ。
GBM系モデルは非線形関係を自動で学習するため、生の統計量を大量に投入することも有効。

**特徴量候補:**
- 全12バンドそれぞれの: 平均、中央値、標準偏差、最小、最大、10/25/75/90パーセンタイル
- バンド間の相関（例: B8 × B11 の相互作用）

## 実装優先度

| 優先度 | 特徴量 | 仮説 |
|--------|--------|------|
| ★★★ | NDBI, 都市化面積率 | 都市密度 |
| ★★★ | NDVI 統計値、低NDVI率 | 植生逆指標 |
| ★★★ | VIIRS avg_rad 統計値 | 経済活動 |
| ★★★ | 全バンド生統計値（mean, std, pct） | 汎用特徴 |
| ★★☆ | MNDWI, 水域率 | 地盤リスク |
| ★★☆ | BSI, 裸地率 | 建設需要 |
| ★☆☆ | Clay Ratio, Iron Oxide | 地質 |
| ★☆☆ | B1 エアロゾル指標 | 工業集積 |
| ★☆☆ | GLCM テクスチャ | 都市構造 |

---

## 実装方針

1. **まず全バンドの基本統計量（mean, std, percentiles）を抽出**してベースラインを作成
2. **NDBI・NDVI・MNDWI・BSI** などの主要指数を計算して追加
3. **VIIRS の空間統計** を加える
4. 特徴量重要度を確認しながら地質・テクスチャ系を追加するか判断する

各TIFはリージョン全体の合成画像（四半期中央値）なので、
ピクセル単位ではなく **画像全体の統計量** を特徴量として使用する。
